Adds the ability to embed a Facebook album and display its photos

== Description ==

The Album extension adds the ability to display photos from a specific Facebook album based on its ID. Display posts in a single column or in a grid layout.


== Installation ==
1. Install the plugin via the WordPress plugin directory, or by uploading the files to your web server (in the /wp-content/plugins/ directory)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > License to enter and activate your license
4. Once activated you can use the 'album' shortcode option to embed your album
