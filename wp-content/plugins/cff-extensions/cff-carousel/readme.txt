== Description ==

The Carousel Feed extension allows you to create a sliding carousel out of your Facebook content.

== Installation ==
1. Install the plugin via the WordPress plugin directory, or by uploading the files to your web server (in the /wp-content/plugins/ directory).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings > License to enter and activate your license
4. Once activated the extension will add settings to the plugin's "Post Layout" page which will allow you to create carousels
