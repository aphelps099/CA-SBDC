== Description ==

Adds the ability to display posts from a specific date range. Once activated the extension will add an option to your Settings page which allows you to display posts from a specific date range. It will also allow you to use the 'from' and 'until' options in your shortcodes.


== Installation ==
1. Install the plugin via the WordPress plugin directory, or by uploading the files to your web server (in the /wp-content/plugins/ directory).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings > License to enter and activate your license
4. Once activated the extension will add an option to your Settings page which allows you to display posts from a specific date range.
